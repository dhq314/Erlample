# Erlample（Erlang Example）

An Erlang Example Repository


## Dependencies

- Python
- PostgreSQL


## Launch Web Server

    python ./erlple.py
    
    
## Generate Erlang Example HTML

	python ./create_erlple.py


## Import erlample.sql

Create Database

    createdb erlample -O erlample -E UTF8 -e

Import erlample.sql to your database.
