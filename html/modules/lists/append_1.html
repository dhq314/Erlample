lists:append/1

附加连接列表集里的每一个子列表

用法：

append(ListOfLists) -> List1

内部实现：

%% append(L) appends the list of lists L

-spec append(ListOfLists) -> List1 when
      ListOfLists :: [List],
      List :: [T],
      List1 :: [T],
      T :: term().

append([E]) -> E;
append([H|T]) -> H ++ append(T);
append([]) -> [].

把 ListOfLists 里的每一个子列表按顺序附加连接成一个新列表

lists:append([[1, 2, 3], [a, b], "just a test"]).
