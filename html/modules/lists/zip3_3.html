lists:zip3/3

把 3 个列表合成一个由 3 个值组成的元组列表

用法：

zip3(List1, List2, List3) -> List4

内部实现：

-spec zip3(List1, List2, List3) -> List4 when
      List1 :: [A],
      List2 :: [B],
      List3 :: [C],
      List4 :: [{A, B, C}],
      A :: term(),
      B :: term(),
      C :: term().

zip3([X | Xs], [Y | Ys], [Z | Zs]) -> [{X, Y, Z} | zip3(Xs, Ys, Zs)];
zip3([], [], []) -> [].

把 3 个列表合成一个由 3 个值组成的元组列表

lists:zip3([a1, a2, a3], [b4, b5, b6], [c7, c8, c9]).
