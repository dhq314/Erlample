application:loaded_applications/0

获取当前已加载过的应用

用法：

loaded_applications() -> [{Application, Description, Vsn}]

返回一个被函数 application:load/1，application:load/2 加载的相关应用信息的列表，也包括被包含而加载进来的应用。Application 是应用名，Description 和 Vsn 分别是它的应用描述信息里的 description 和 vsn 的键值。
