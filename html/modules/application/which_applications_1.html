application:which_applications/1

获取当前正在运行的应用程序信息的列表

用法：

which_applications(Timeout) -> [{Application, Description, Vsn}]

返回一个当前正在运行的应用程序信息的列表。Application 是应用程序的名字。Description 和 Vsn 分别是应用程序描述信息和指定的版本值。参数 Timeout 是该函数操作的超时值。

application:which_applications(5000).
