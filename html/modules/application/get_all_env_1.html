application:get_all_env/1

获取一个应用程序的所有配置参数

用法：

get_all_env(Application) -> Env

获取指定应用程序 Application 的所有配置参数信息。如果指定应用程序 Application 没有被加载，那么函数则返回 []。

{ok, Application} = application:get_application(),
application:get_all_env(Application).
