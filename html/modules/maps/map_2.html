maps:map/2

对映射组里的值循环执行一系列操作并返回一个新映射组

用法：

map(Fun, Map1) -> Map2

对映射组 Map1 里的每一个键值调用函数 F(K, V1)，并返回原映射组里每一个键值调用后所产生的一个新映射组 Map2。在新映射组 Map2 里，调用函数 F/2 所返回的新值是对应原来的值的键的。

Fun = fun(K,V1) when is_list(K) -> V1*2 end,
Map = #{"k1" => 1, "k2" => 2, "k3" => 3},
maps:map(Fun, Map).
