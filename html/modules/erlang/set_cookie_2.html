erlang:set_cookie/2

给一个节点设置 cookie

用法：

erlang:set_cookie(Node, Cookie) -> true

内部实现：

-spec erlang:set_cookie(Node, Cookie) -> true when
      Node :: node(),
      Cookie :: atom().
set_cookie(Node, C) when Node =/= nonode@nohost, erlang:is_atom(Node) ->
    case erlang:is_atom(C) of
	true ->
	    auth:set_cookie(Node, C);
	false ->
	    erlang:error(badarg)
    end.

将节点 Node 的 cookie 设置为原子 Cookie。如果 Node 是本地节点，那么函数也会设置其他未知节点的 cookie 为 Cookie。

Node = erlang:node(),
Cookie = erlang:get_cookie(),
erlang:set_cookie(Node, Cookie).

如果本地节点不活跃了，将会报一个 function_clause 的错误。
