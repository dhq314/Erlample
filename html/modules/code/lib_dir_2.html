code:lib_dir/2

返回一个应用的子目录

用法：

lib_dir(Name, SubDir) -> file:filename() | {error, bad_name}

返回一个应用的根目录下的一个子目录路径。

通常情况下，子目录一般放在应用的根目录下，但是如果至少有一部分应用放在存档里时，情况就不一样了。一些子目录可能放在普通的目录里，而另外的则放在存档文件里。

如果目录真的存在，则不会做检测操作。

{ok, Application} = application:get_application(),
code:lib_dir(Application, priv).
