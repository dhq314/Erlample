code:is_module_native/1

检测模块是有有原生代码

用法：

is_module_native(Module) -> boolean() | undefined

如果一个已经加载的模块 Module 里有原生代码加载，那么该函数返回 true，如果模块已经加载，不过没有原生代码，则返回 false。如果模块没有加载，那么函数则返回 undefined。
