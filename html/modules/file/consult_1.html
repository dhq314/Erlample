file:consult/1

把文件里的 Erlang 项值读出来

用法：

consult(Filename) -> {ok, Terms} | {error, Reason}

从文件 Filename 里读取由 '.' 分割的 Erlang 项。返回的结果有：

  • {ok, Terms}：成功读取文件里的 Erlang 项。
  • {error, atom()}：打开或读取文件的时候发生了一个错误。详细的错误码可查看 file:open/2 里的介绍。
  • {error, {Line, Mod, Term}}：解析文件里的 Erlang 项时发生了一个错误。可以使用 file:format_error/1 把这个三元素元组（hree-element tuple）的错误标示转为一种更直观的描述。
file:consult("/app/rebar.config").

文件的编码可以用下面的注释形式来设置：

%% coding: utf-8

%% For this file we have chosen encoding = Latin-1

%% -*- coding: latin-1 -*-
