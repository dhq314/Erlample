file:change_time/3

更改一个文件的最后访问和修改时间

用法：

change_time(Filename, Atime, Mtime) -> ok | {error, Reason}

更改一个文件 Filename 的最后访问时间 Atime 和修改时间 Mtime。

Time = erlang:localtime(),
file:change_time("./rebar.config", Time, Time),
file:read_file_info("./rebar.config").
