file:sync/1

同步数据到磁盘

用法：

sync(IoDevice) -> ok | {error, Reason}

确保操作系统（不是 Erlang 的运行系统）缓冲区里的数据全部写入磁盘。在一些平台，这个函数可能没有效。

一些典型的错误原因：

  • enospc：没有足够的磁盘空间剩余来写入文件。
{ok, IoDevice} = file:open("./rebar.config", read),
file:sync(IoDevice).
